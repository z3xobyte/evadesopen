Features

The 9113 number button allows you to set the chrono "shadow" to any time, the bigger the number the futher back in time chrono will backtrack (same with enemies and rewind)

Hotkeys

(H) - Hide hero card

(M or Tab) - Hide minimap

(E) - Go back 1 by area (cheating enabled)

(R) - Go forward by 10 areas (cheating enabled)

(T) - Go forward by 1 area (cheating enabled)

(v) - Invincibility (cheating enabled)

(B) - Removes cooldown on abilities (cheating enabled)

(N) - Activates wall god, where you can now go out of bounds (cheating enabled)

([) - Adds a safe point. Upon death you will respawn at the position that you last pressed [ (Test Mode only)

(]) - Removes the most recent safe point (Test Mode only)

(P) - If true and a safe point exists, upon death it will reset the timer (Test Mode only)

(O) - Resets timer (Test Mode only)
